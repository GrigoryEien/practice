\subsection*{Базовый}
\begin{enumerate}
    \item Пусть есть монетки, на одной стороне которых написано $1$, на другой $2$. 
    Пусть $\xi$ - ДСВ суммы результатов бросков 2х монеток, $\mu$ - проиведение. 
    Найдите медиану, матожидание случайных величин и сравните их между собой.
    \item Докажите c помощью определений свойства мат.ожидания, дисперсии, ковариации.
    \begin{enumerate}
        \item $E(\alpha \xi) = \alpha E(\xi)$
        \item $E(\xi + \mu) = E\xi + E\mu$
        \item $E(\alpha\xi + \beta\mu) = \alpha E\xi + \beta E\mu$
        \item $E(\xi\mu) = E\xi E\mu$ если независимы
        \item Матожидание в схеме бернулли равно $np$.
    \end{enumerate}
    \item Докажите свойство  $D(\alpha \xi) = \alpha^2 D(\xi)$
    \item В каком случае $D(\xi + \mu) = D\xi + D\mu$? А в каком не равно? На какую величину они будут отличаться в общем случае?
    \item Докажите свойства ковариаций
    \begin{enumerate}
        \item $cov(\xi, \mu) = cov(\mu, \xi)$
        \item $cov(\alpha\xi, \mu) = \alphacov(\xi, \mu)$
        \item $cov(\xi_1 + \xi_2, \mu) = cov(\xi_1, \mu) + cov(\xi_2, \mu)$
    \end{enumerate}
    \item 
Двумерное распределение пары целочисленных случайных величин $\xi$ и $\eta$ задаётся с помощью таблицы

\end{center}
где в пересечении столбца $\xi = i$ и строки $\eta = j$ находится вероятность $P\lbrace{\xi = i, \eta = j\rbrace}$. Найти:

    \item Среднеквадратичное отклонение: $\sigma(\xi,\eta)$
